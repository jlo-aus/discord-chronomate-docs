Permissions

  • Manage Server
  • View Channels
  • Send Messages
  • Manage Messages

Manage Server is needed to change the server icon.
View channels is need to view see the icons in the icon channel.
Send Messages is needed to send bot responses to commands.
Manage messages is needed to auto remove some bot responses after set time.
Add Reactions is needed to create paginated lists using reactions to switch pages.
