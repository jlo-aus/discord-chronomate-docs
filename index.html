Automate your Discord server icon with calendar based scheduling

Get StartedInvite Chronomate

Storage Channel

Keep icons stored where they are used - on Discord. Change the icon at any time to any of the stored icons. Set the channel as public to allow other users to contribute their own icons.

Scheduled Icon Changes

Schedule icon changes for any date on the calendar and Chronomate will take care of the rest.

Timezone Aware

Chronomate uses Coordinated Universal Time (UTC) by default. Change the timezone to best match you and your Discord server.
